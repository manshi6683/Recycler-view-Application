# 📌 About Project

**MyRecyclerViewApp** is a modern Android application built using **Kotlin** and **RecyclerView**.

The app displays stylish coffee cards with:

- ☕ Coffee images
- 📝 Coffee names
- 💬 Descriptions
- 🎨 Modern dark UI
- 📱 Smooth scrolling RecyclerView

This project is designed to practice:

- RecyclerView
- Adapter & ViewHolder
- CardView
- Custom Layout Design
- Kotlin Android Development

```text

     ↓

     ↓

     ↓
Coffee Model Data
```

# 💻 Important Components

## 📌 RecyclerView

Used for displaying scrollable coffee items efficiently.

---

## 📌 Adapter

`CoffeeAdapter.kt`

Responsible for:

- Binding data
- Inflating layouts
- Connecting RecyclerView with data

---

## 📌 Model Class

`Coffee.kt`

Stores coffee item data such as:

```kotlin
Coffee(
    image,
    title,
    description
)
```
